06Recommended New File Structure — Option A (Feature-co-located)

Each page owns its sub-components, hooks, and types. Shared entities (used by 2+ pages) graduate to src/types/ and src/services/.

07Design Patterns — Existing Code to Follow
Stat Card (Dashboard)
className="bg-white rounded-2xl p-6 shadow-sm border"

Use for all KPI cards. Grid wrapper: grid grid-cols-N gap-5.

Orange Accent — only accent color
isActive ? "bg-orange-50 text-orange-500" : "text-gray-600 hover:bg-gray-100"

Do not introduce blue or green as primary action colors. Orange is the sole accent.

Page Content Padding
MainLayout already wraps content in: className="p-6"

Pages must NOT add their own outer padding. Only add internal spacing (e.g., mb-6 for headings).

Ant Design + Tailwind split
antd → Table, Modal, Form, Select, DatePicker, Tabs tailwind → layout, spacing, color, flex/grid

Both are installed. Use Ant Design for data-heavy UI; Tailwind for everything else.

Form Pattern
useForm({ resolver: zodResolver(schema) }) + antd Form.Item

@hookform/resolvers and zod are both installed. Use this stack for all modals and forms.

Avatar (Header)
className="w-10 h-10 rounded-full bg-purple-600 text-white flex items-center justify-center"

Purple is reserved for user avatars only. Do not repurpose as an accent.
